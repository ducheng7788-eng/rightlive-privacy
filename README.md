# RightLive Translate Privacy Policy / 隐私政策（Free 1.0）

Last updated / 更新日期: 2026-05-19

This policy explains how the **RightLive Translate** Chrome extension handles data in the current free version.
本政策说明 **RightLive Translate** Chrome 扩展在当前免费版本中的数据处理方式。

## 1) Single Purpose / 单一用途

- The extension's single purpose is to provide real-time webpage text translation and side-by-side reading.
- 本扩展的单一用途是提供网页文本实时翻译与原文/译文对照阅读。

## 2) What Data Is Processed / 处理哪些数据

- Visible webpage text that the user chooses to translate (manual click, hover, or selected text/paragraph).
- 用户触发翻译时的网页可见文本（手动点击、悬停、选词/选段）。

## 3) What Data Is Stored Locally / 本地存储哪些数据

Stored in Chrome extension local storage:
保存在 Chrome 扩展本地存储中的数据：

- UI preferences (theme, UI language, target language, panel size/position)
- 界面偏好（主题、界面语言、目标语言、面板尺寸/位置）
- Feature switches (pause, hover mode, scroll sync, privacy mode)
- 功能开关（暂停、悬停模式、滚动同步、隐私模式）
- Translation cache and notebook entries created by the user
- 翻译缓存与用户创建的生词本内容
- Daily free quota counters
- 免费额度计数信息

## 4) Data Sent to Third Parties / 发送到第三方的数据

To return translation results, the extension sends selected webpage text to:
为返回翻译结果，扩展会把用户触发翻译的文本发送到：

- `https://translate.googleapis.com/`
- Dictionary lookup endpoint (when needed): `https://api.dictionaryapi.dev/`

Transmission uses HTTPS.
传输使用 HTTPS 加密。

## 5) Data We Do NOT Collect / 不收集的数据

The extension does not intentionally collect:
本扩展不会主动收集以下数据：

- Personal identity data (name, email, phone, ID number)
- 个人身份信息（姓名、邮箱、手机号、证件号）
- Financial/payment data
- 财务与支付信息
- Authentication secrets (passwords, PINs)
- 身份验证信息（密码、PIN）
- Health data
- 健康信息
- Contact lists, private files, photos, audio/video files
- 通讯录、私有文件、照片、音视频文件

## 6) Ads, Sale, and Sharing / 广告、出售与共享

- No advertising use.
- 不用于广告投放。
- No sale of user data.
- 不出售用户数据。
- No sharing with data brokers.
- 不与数据经纪商共享数据。
- Data is used only to provide translation functionality.
- 数据仅用于提供翻译功能。

## 7) User Control / 用户可控性

Users can:
用户可以：

- Turn the extension off at any time
- 随时关闭插件
- Clear translation cache
- 清除翻译缓存
- Clear notebook entries
- 清空生词本
- Remove the extension to stop all future processing
- 卸载扩展以停止后续数据处理

## 8) Sensitive Pages / 敏感页面

For login/account-related pages, users should use manual translation carefully.
在登录/账号相关页面，建议用户谨慎使用手动翻译功能。
